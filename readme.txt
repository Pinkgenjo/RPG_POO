# Instruções para execução do jogo

1. Abra o terminal no Visual Studio Code.
2. Navegue até a pasta do projeto.
3. Execute o seguinte comando para iniciar o jogo:

   node index.js

Certifique-se de que o Node.js esteja instalado na sua máquina.

# Comandos disponíveis no jogo

- mover [direcao]           → Move o jogador para uma sala vizinha na direção especificada.
- pegar [nome_do_item]      → Coleta um item da sala atual.
- usar [nome_do_item]       → Usa um item do inventário.
- inventario                → Exibe os itens no inventário do jogador.
- ajuda                     → Mostra a lista de comandos disponíveis.
- sair                      → Encerra o jogo.
